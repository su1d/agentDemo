# 多智能体协调系统设计方案

## 概述

基于现有架构（Java 21 Spring Boot + Python LangGraph + Vue 3）扩展为多智能体系统。
同质 Agent 分工模式 + Java 后端统一调度。

## 架构

```
用户 (Vue 3, port 5173)
    ↓ HTTP
Java 调度器 (Spring Boot, port 8080) ←→ 每个 Agent 一个 Python Worker 进程
    │                                    ├── Agent-A: 计算/数学 (calculator)
    │                                    ├── Agent-B: 天气/地理 (weather)
    │                                    ├── Agent-C: 总结/输出 (summarizer)
    │                                    ├── Agent-D: 搜索 (searcher)
    │                                    └── Agent-E: 通用对话 (chat)
    │
    ├── /api/chat          ← 入口，自动路由到合适 Agent
    ├── /api/agents        ← 查看所有 Agent 状态
    └── /api/chat/orchestrate ← 手动指定多个 Agent 协作
```

## 通信协议

每个 Worker 进程使用现有的 stdin/stdout JSON 协议。
Java 端通过 `AgentWorkerManager` 管理所有 Worker 的生命周期。
